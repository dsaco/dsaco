### tooltip.scss

```
npm i -S @dsaco/dscss
```

```
$ds-tooltip-bg: pink;
$ds-tooltip-color: yellow;

// $ds-tooltip-size: 8px !default;
// $ds-tooltip-bg: rgba(0, 0, 0, 0.75) !default;

@import "@dsaco/dscss/tooltip";
```
